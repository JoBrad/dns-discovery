## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| local config file -> process | Untrusted file content is parsed into runtime settings |
| CLI flags -> config resolution | User-provided paths and overrides alter process behavior |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | T | internal/config/config.go | mitigate | Parse with `encoding/json`, validate supported fields, and reject malformed or empty domain entries before use |
| T-03-02 | D | cmd/dns-discovery/main.go | mitigate | Fail fast on unreadable config paths and bound config handling to startup before network activity |
| T-03-03 | I | error output | accept | Config path appears in errors because the tool is local CLI software and path disclosure is low risk |
</threat_model>
